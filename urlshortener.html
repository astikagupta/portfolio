URLShortener : Shorten any URL


User Interface Design | Front End Development | Python Programming | Prototyping


My Role

Individually designed and coded a URL Shortener web application that shortens any given URL to the user provided text using Javascript, HTML5, CSS3 and Python.

Description

I created a URL shortener that basically takes an actual long URL as input from the user and allows the user to enter any name (could be a single or multiple words) for this URL and maps this entry with the actual URL. So, the user just by putting the shorter name, would be taken to the actual URL page.In this way this functionality shortens the actual longer URL to something that the user selects and so it is easier for him to access his favorite pages easily instead of putting in the longer URLs every time.
